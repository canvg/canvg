# Options Presets

There are two options presets:

- [`presets.offscreen()`](/api/namespaces/presets#offscreen): options for [`OffscreenCanvas`](https://developer.mozilla.org/en-US/docs/Web/API/OffscreenCanvas) ([example](/examples/offscreen))
- [`presets.node({ DOMParser, canvas, fetch })`](/api/namespaces/presets#node): options for NodeJS with [`node-canvas`](https://github.com/Automattic/node-canvas) ([example](/examples/nodejs))
